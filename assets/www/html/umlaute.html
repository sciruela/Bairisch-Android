Umlaute

  • Umlaute:

  • ä → e Schäferhund → Schefahund

  • ä → a Käse → Kaas

  • äl → äi schälen → schäin

  • ö → e mögen → megn, schön → schee

  • ö → i König → Kini, können → kinna

  • öl → äi Hölle → Häi

  • ü → i Schlüssel → Schlissl, dünn → dinn

  • ü → u bücken → bugga, zurück → zruck, Küche → Kuchl

  • ü → ia müde → miad, grüßen → griaßn

  • ü → ea grün → grea, Blümchen → Bleamal

  • ül → ui bzw. äi Gefühl → Gfuih bzw. Gfäi, Mühle → Muih bzw. Mäih

  • Wort mit ü → Bairisch: neues Wort

  • Rücken → Buggl

  • verrückt → damisch

  • pflücken → brogga
